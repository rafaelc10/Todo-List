## :computer: Projeto
Projeto realizado com o objetivo de aprimorar e testar os conhecimentos em ReactJS.

#### New ⭐
- Tarefas salvas com LocalStorage.
- Estilização com TailwindCSS.
#### Instalação
- `git clone https://github.com/rafaelc10/Todo-List`
- `npm install`
#### Para executar
- `npm run dev`
